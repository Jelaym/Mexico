Módulo Cuestionario (Alumno)



Esta sección de la página se utiliza cuando se quiera avanzar de época en la sección "Mapa de México"; Una vez que se halla interactuado con la información que proporciona el mapa, se puede ir a otra época, pero para esto el alumno deberá contestar un Cuestionario que se desplega una vez que se halla seleccionado la época siguiente.


Para poder avanzar a la siguiente época se requerirá obtener como mínimo una puntuación de 7 en el cuestionario

El alumno no puede modificar la información de este módulo
